    SCSS

    A reusable SCSS library

    Install

    npm i git+https://github.com/mayognaise/scss.git

    Usage

    Default

    You can simple call:

    @import 'scss';

    Customize variables

    Here is the default variables.

    If you wish customize variables:

    // common.scss - Create a common scss file to use
@import 'path/to/your/own/variables'; // Import customized variable file. Don't forget adding
@import 'scss';

// You can use this file as library
@import 'path/to/common';

    Layout

    mixins

    max-width-center

    @mixin max-width-center($max-width: $grid-width, $margin-top: 0, $margin-bottom: 0) { ... }

    Description

    Centre a block to make it a container. Default value of $grid-width is here.

    Parameters

    parameter Nameparameter Descriptionparameter Typeparameter Default value
    $max-width

    Max width for the container.

    		Number$grid-width
    $margin-top

    Top margin.

    		Number0
    $margin-bottom

    Bottom margin.

    		Number0

    Example

    @include max-width-center;
//   margin: 0 auto 0;
//   max-width: 960px;
//   width: 100%;
    @include max-width-center(1200px);
//   margin: 0 auto 0;
//   max-width: 1200px;
//   width: 100%;
    @include max-width-center(null, 30px);
//   margin: 30px auto 0;
//   max-width: 960px;
//   width: 100%;
    @include max-width-center(null, null, 30px);
//   margin: 0 auto 30px;
//   max-width: 960px;
//   width: 100%;

    Output

    max-width, margin and width

    Requires

    • [function] is-unitless
    • [function] is-unitless
    • [function] is-unitless
    • [variable] grid-width

    x-small

    @mixin x-small() { ... }

    Description

    Mixin to add media query for x-small breakpoint (such as iPhone 5/SE). Default is map-get($breakpoints, 'x-small'). To update the key, update $breakpoints

    Parameters

    None.

    Example

    @include x-small {
  color: red;
}
// @media (max-width: 359px) {
//   color: red;
// }

    Requires

    • [mixin] respond-to
    • [variable] breakpoints

    is-unitless

    @function is-unitless($value) { ... }

    Description

    Return if value is numbrable after removing unit

    Parameters

    parameter Nameparameter Descriptionparameter Typeparameter Default value
    $value

    Number to remove unit from

    		Number or String— none

    Returns

    Boolean

    Example

    $value: is-unitless(10px);
// -> true
    $value: is-unitless(10);
// -> true
    $value: is-unitless(hello);
// -> false

    Requires

    • [function] strip-units

    Used by

    • [mixin] max-width-center
